## 🎯 What is JavaVerse 2.0?

> **A zero-setup, browser-based Java interview preparation platform.** 205+ curated questions from Arrays, Strings, Logic, and Dynamic Programming — with a built-in live Java compiler to write, run, and test your solutions in real time.

**Built for:**
- 🎓 Java developers cracking FAANG & product-based company interviews
- 📚 Students mastering DSA fundamentals with hands-on practice
- 💡 Anyone who wants to go from theory to working code instantly
- 🏢 Interview preppers who can't afford slow LeetCode compile times

| Layer | Technology | Why |
|:---|:---|:---|
| **UI** | Pure HTML5 + CSS3 | No framework overhead — instant load, zero bundle size |
| **Logic** | Vanilla JavaScript | No dependencies, no breaking updates |
| **Compiler** | Live Java Compiler (Browser API) | Run Java code without installing JDK |
| **Animations** | CSS keyframes + Canvas | 3D effects, smooth transitions, particle effects |
| **Hosting** | Vercel | Global CDN, instant deploys, always-on |
| **Theme** | CSS variables | Clean light/dark toggle with persistence |
